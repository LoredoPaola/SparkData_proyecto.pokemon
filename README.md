# 📌 SparkData: Enciclopedia Pokemon 

<img width="2752" height="1536" alt="projectbanner" src="https://github.com/user-attachments/assets/4947d17a-fc90-4cc7-aad9-36727ecb8abe" />

El equipo de Spark Data se complace en compartir el resultado final de su proyecto Enciclopedia Pokémon donde se pueden apreciar gráficas demostrativas del contenido de los datos proporcionados limpios y optimizados.

## 🗂️ Descripción

Este proyecto contiene información sobre los personajes de pokemon de 7 generaciones almacenada en una base de datos ordenada y optimizada para la búsqueda y análisis de datos. A través de estos datos se generaron 4 diferentes gráficas que ayudan a visualizar el contenido de estos.
